# CMaker

This gem generate  source files needed for CMakeLists.txt

## Installation

Add this line to your application's Gemfile:

    gem 'cmaker'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cmaker

## Usage
  
    $ cd /path/to/your/project
    $ cmaker config
    $ cmaker
  
you will get a cmakelists.txt /path/to/your/project.
